API CHEAT SHEET

Zanimo(elt | promise[elt])
➜ promise[elt]

Takes a dom element or a promise of a dom element and returns a promise of it.

Zanimo.transition(
    elt,
    property,
    value,
    duration,
    [timingƒ]
) ➜ promise[elt]

Start a CSS `property` transition with value `value` on the dom element `elt` with a duration of `duration` ms and an optional timing function `timingƒ`.

Zanimo.transitionf(
    property,
    value,
    duration,
    [timingƒ]
) ➜ (elt) ➜ promise[elt]

Wrap `Zanimo.transition` into a function; useful in a chain of promises.

Zanimo.transform(
    elt,
    value,
    [overwrite]
) ➜ promise[elt]

Set CSS transform property of dom element `elt` to `value` and return a promise of the dom element. Boolean argument `overwrite` allows to overwrite present transform property, defaults append.

Zanimo.transformf(
    value,
    [overwrite]
) ➜ (elt) ➜ promise[elt]

Wrap `Zanimo.transform` into a function; useful in a chain of promises.

Zanimo.all(
    array[ (elt) ➜ promise[elt] ]
) ➜ (elt) ➜ promise[elt]

Allows running multiples transitions on the same dom element at the same time.

Zanimo.f(elt | promise[elt])
➜ (elt) ➜ promise[elt]

Wrap `Zanimo()` into a function; useful in a chain of promises;
